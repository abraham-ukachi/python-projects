## Le Pendu

### Installation

```sh
git clone https://github.com/abraham-ukachi/python-projects.git && cd python-projects
```

### Example usage

```sh
python3 pendu.py
```
> NOTE: 

```python3
from pendu import PenduGame

pendu = PenduGame('en')
pendu.requestPlayerLevel()
pendu.startGame()
```
> NOTE: **'en'** stands for _English_. Use **'fr'**, to play the game in _French_
